Проект устроен следующим образом:
На сцене находится объект с компонентом Bootstrapper, который является входной точкой для всего проекта.
В методе Start у Bootstrapper сначала инициализируется DIContainer, а следом за этим запускается игра через класс GameManager.
Зависимости регистрируются  в скрипте DIInstaller, который также висит на сцене на объекте Bootstrapper.

Сам проект состоит из двух логических частей - мета игра и геймплей.
Каждая часть в свою очередь разделена на Model/View/Controller, где и описывается поведение игры в каждом конкретном логическом сегменте.

Также есть минимальный набор сервисов для сохранения данных, управления прогрессом игрока и внедрением зависимостей (DI Container).

В разработке активно принимал участие ИИ в лице его величества Gemini 3.1 Pro.


Проект запускается и работает с единственной сцены в игре MainScene.
